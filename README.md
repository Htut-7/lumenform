## Features

- Live 3D product catalog — six lighting pieces, each rendered
  procedurally and rotatable in the browser
- Interactive product viewer — drag to orbit, scroll to zoom
- Firebase Authentication — email/password and Google sign-in
- Persistent cart — synced to Firestore per signed-in user
- Order history — recorded to Firestore on checkout
- Locked-down Firestore rules — every user can only read/write their own data
